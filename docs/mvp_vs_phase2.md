# MVP vs Phase-2 Scope

## MVP

Focus: reliable execution copilot.

Includes:

- intent resolver
- situation detector
- autonomy guard
- agent orchestration
- skill system
- tool routing
- browser automation
- desktop actions
- external runtime adapter
- confirmation loop
- local memory
- CLI interface

No avatar.
No emotion model.
No 3D UI.

---

## Phase-2

Focus: embodiment and interaction.

Adds:

- avatar engine
- emotion model
- voice IO
- continuous listening
- richer context sensing
- behavior adaptation
- trust model tuning
- visual canvas control
- multi-device sync

---

## Phase-3 (Optional)

- 3D avatar
- realtime scene awareness
- gesture + presence layer
